# Challenge rules (v1) — progress points + Fortachones updates

## Core
1. Weeks run **Monday–Sunday**.
2. **Progress points** toward the weekly goal of 5:
   - Each distinct calendar day with ≥1 valid workout = **+1**
   - First day with 2+ workouts (**double day**) adds **+1 bonus**
   - Extra double days the same week do **not** add another bonus
   - Example: 1 prior day + a double day = **3** points; **4 days + 1 double = 5/5**
3. Missed days = `max(0, 5 - progress_points)`.
4. When progress exceeds 5 (typically **6/5** via double):
   - If the user still has unpaid prior failed days → clear **1** prior failed day
   - Else → bank **+1 día a favor** (credit) for future weeks
5. Banked “días a favor” spend against later missed days when weeks close.
6. Fee = **$100 MXN** per final missed day.
7. Photo evidence required; visible to the group.
8. Group money pool is called **guardadito** (not “pozo”).

## Grace period / timezones
Weeks stay **open for 2 days after Sunday** (close on Tuesday). Logging/adjustments allowed for today or up to 2 days back, but never into a locked week. This buffers timezone differences across the crew.

## Groups
- Admins create the group and share an **invite code / deep link**.
- Only admins can add/remove members.
- Removing a member drops their owed amount from the **active group guardadito**.

## Units
Metric: **kg** and **meters**.
